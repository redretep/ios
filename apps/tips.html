Tips & FAQ

What to do when "This page can't be displayed" or "Refused to connect" appears?

This happens because modern platforms block themselves from running inside security frames (iFrames).

Solution: Please install the Ignore X-Frame-Headers extension in your web browser to seamlessly access these configurations.

Can I run .ipa files?

No. You can only interact with web ecosystems here. This sandbox is built entirely as a visual frontend simulation, not an actual emulated iOS kernel.

How can I add an app?

On Desktop Devices:

  1. Right-click anywhere on the empty layout workspace.
  2. Select Add Web App from the context window.
  3. Type your desired application name.
  4. Input the destination web address and save.

On Mobile Devices:

  1. Long-press anywhere on the background for 1 second until the workspace wiggles.
  2. Tap the + (Plus) action engine at the top.
  3. Search or specify your workspace app targets.
  4. Input your destination address.
How can I send apps / wallpapers to another device?

Open up the main Settings module interface, select the system options, and choose to export your configuration profile matrix to a localized .json layout package file.

Dispatch that specific package file to your companion ecosystem via a reliable online transfer service (like fileditch.github.io), then trigger the internal workspace configuration load utility on the targeted hardware platform.

Can I add custom widgets?

Right now, no. Personalized dynamic home screen applet widgets are currently non-functional within this development roadmap phase.
